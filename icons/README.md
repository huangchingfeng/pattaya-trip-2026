# PWA 圖示說明

## 🎨 圖示需求

PWA 需要以下圖示：
- **icon-192.png** (192x192 像素)
- **icon-512.png** (512x512 像素)

## 🚀 快速生成方法

### 方法 1：使用 generate-icons.html（推薦）

1. 開啟瀏覽器訪問 `/generate-icons.html`
2. 點擊「下載」按鈕取得圖示
3. 將檔案放到 `/icons/` 資料夾

### 方法 2：線上工具

使用 [favicon.io](https://favicon.io/favicon-generator/) 生成圖示：

**設定建議**：
- 文字：P (Pattaya 首字母)
- 背景：漸層（#ff00ff → #00ffff）
- 字型：粗體
- 形狀：圓角正方形

### 方法 3：使用 Emoji

最簡單的方法：
- 使用 🏖️ (沙灘) 或 🌴 (椰子樹) emoji
- 背景：霓虹粉紅到藍色漸層

## 📐 圖示規格

- **格式**：PNG
- **尺寸**：192x192, 512x512
- **背景**：不透明（避免顯示問題）
- **主題**：芭達雅旅遊（沙灘、夕陽、派對）

## ⚠️ 臨時方案

如果沒有圖示，PWA 仍可運作，但：
- Android 安裝時會顯示警告
- iOS 會使用截圖作為圖示

**建議**：盡快補上圖示以提升專業度
