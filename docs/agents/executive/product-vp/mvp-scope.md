# MVP 範圍定義
**MVP Scope Definition**

## 🎯 MVP 核心定義

### 什麼是 MVP？

> **Minimum Viable Product (最小可行產品)** = 用最少的功能，驗證最核心的價值

### Pattaya PWA 的 MVP 目標

**核心價值**：讓用戶可以在**沒有網路的情況下**瀏覽芭達雅旅遊攻略

**驗證假設**：
1. 用戶在泰國旅遊時會遇到網路不穩的問題
2. 離線可用的旅遊攻略有實際需求
3. PWA 安裝比傳統網頁更方便

---

## ✅ MVP 包含功能

### 1. 核心 PWA 功能

| 功能 | 需求 | 驗收標準 |
|------|------|---------|
| **可安裝性** | manifest.json + meta tags | 可安裝到 Android/iOS 主畫面 |
| **離線瀏覽** | Service Worker 快取 | 9 個核心頁面離線可用 |
| **泰語助手** | 搜尋功能離線化 | 無網路也能搜尋泰語單字 |
| **快速啟動** | 預快取核心資源 | 啟動時間 < 1 秒 |

### 2. 離線頁面清單（9/11 頁）

**必須離線可用**：
1. ✅ `index.html` - 首頁導航
2. ✅ `day.html` - 白天行程
3. ✅ `night.html` - 夜生活攻略
4. ✅ `itinerary.html` - 行程總覽
5. ✅ `thai-helper.html` - 泰語助手
6. ✅ `sop.html` - 旅遊 SOP
7. ✅ `money.html` - 換錢資訊
8. ✅ `esim.html` - eSIM 資訊
9. ✅ `games.html` - 派對遊戲

**暫時排除**：
- ❌ `resources.html`（主要是外部連結，需要網路）
- ❌ `truth-or-dare.html`（非核心功能）

### 3. Service Worker 功能

**必備功能**：
- ✅ `install` event - 預快取核心資源
- ✅ `fetch` event - 攔截請求並回應快取
- ✅ `activate` event - 清理舊快取
- ✅ Network First 策略（fallback to Cache）

**暫時排除**：
- ❌ 複雜的快取策略（Cache First, Stale-While-Revalidate）
- ❌ 推送通知
- ❌ 背景同步

### 4. manifest.json 配置

**必備欄位**：
```json
{
  "name": "芭達雅之旅 2026",
  "short_name": "Pattaya 2026",
  "start_url": "/",
  "display": "standalone",
  "theme_color": "#ff00ff",
  "background_color": "#0a0a0f",
  "icons": [...]
}
```

**暫時排除**：
- ❌ screenshots（非必須）
- ❌ shortcuts（非必須）
- ❌ share_target（進階功能）

### 5. PWA 圖示

**必備尺寸**：
- ✅ 192x192 (any)
- ✅ 512x512 (any)

**暫時排除**：
- ❌ maskable 圖示（iOS 不支援）
- ❌ 多種尺寸（72, 96, 128, 144, 152, 384）

## ❌ MVP 不包含功能

### Phase 2 功能（延後到下一版本）

| 功能 | 排除原因 | 預計納入時程 |
|------|---------|-------------|
| **推送通知** | iOS Safari 不支援 | Phase 4 |
| **背景同步** | 非核心需求 | Phase 4 |
| **IndexedDB** | 無需儲存用戶資料 | Phase 4 |
| **個人化設定** | 增加複雜度 | Phase 4 |
| **多語言** | 目前僅繁中 | 未規劃 |
| **用戶登入** | 無需帳號 | 未規劃 |

### 非必要的 UI/UX 功能

| 功能 | 排除原因 | 是否未來加入 |
|------|---------|-------------|
| **底部導航** | 現有導航已足夠 | 可能 |
| **側邊選單** | 頁面數量少 | 不需要 |
| **滑動手勢** | 增加複雜度 | 可能 |
| **動畫效果** | 非核心體驗 | 可能 |
| **深色模式切換** | night.html 已是深色 | 不需要 |

## ✅ 總結

### MVP 定位

**Pattaya Trip 2026 MVP = 可安裝 + 離線可用的旅遊攻略**

### 核心價值

1. **安裝方便** - 像 App 一樣加入主畫面
2. **離線可用** - 9 個核心頁面完全離線
3. **泰語助手** - 無網路也能查單字
4. **輕量快速** - 啟動時間 < 1 秒

### 不做的事

- ❌ 不做推送通知（iOS 不支援）
- ❌ 不做複雜的個人化（增加複雜度）
- ❌ 不做多語言（目前僅繁中）
- ❌ 不做用戶登入（無需帳號）

**MVP 目標**：用最少的工時（4-7 小時），實現最核心的價值（離線旅遊攻略）
