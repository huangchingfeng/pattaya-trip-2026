# PWA 測試檢查清單

## ✅ 基礎功能測試

### Manifest 檢查
- [ ] 開啟 Chrome DevTools > Application > Manifest
- [ ] 確認名稱顯示為「芭達雅之旅 2026」
- [ ] 確認 start_url 為 `/`
- [ ] 確認 display 為 `standalone`
- [ ] 確認圖示正確顯示（192x192 和 512x512）
- [ ] 確認 theme_color 為 `#ff00ff`

### Service Worker 檢查
- [ ] 開啟 Chrome DevTools > Application > Service Workers
- [ ] 確認 SW 狀態為 **ACTIVATED**
- [ ] 確認 SW 版本為 `v1.0.0`
- [ ] Console 顯示「✅ Service Worker 註冊成功」
- [ ] Console 顯示「[SW] Pre-cache complete」

### 離線功能測試
- [ ] 開啟 Chrome DevTools > Network
- [ ] 勾選「Offline」
- [ ] 重新整理頁面
- [ ] 確認首頁仍可正常顯示
- [ ] 依序測試以下頁面是否可離線瀏覽：
  - [ ] `/` (首頁)
  - [ ] `/day.html` (白天行程)
  - [ ] `/night.html` (夜間行程)
  - [ ] `/itinerary.html` (7天行程)
  - [ ] `/thai-helper.html` (泰語助手)
  - [ ] `/games.html` (派對遊戲)
  - [ ] `/sop.html` (SOP指南)
  - [ ] `/money.html` (換錢攻略)
  - [ ] `/esim.html` (eSIM設定)

### 泰語助手離線測試
- [ ] 切換到離線模式
- [ ] 開啟 `/thai-helper.html`
- [ ] 輸入中文搜尋（例如：你好）
- [ ] 確認搜尋結果正常顯示
- [ ] 點擊發音按鈕（若支援）
- [ ] 確認可瀏覽完整 410+ 句型
